How to Play 2048:

1. Start with a 6x6 grid and two tiles (2 or 4).
2. Use arrow keys to move all tiles.
↑
← ↓ →
3. Merge same-number tiles: 2 + 2 = 4
4. New 2 or 4 tile appears after each move.
5. Combine tiles to reach 2048 and win!
6. Game ends when:
  • You create a 2048 tile (Win!)
  • No more moves possible (Game over)
7. Tips:
  • Keep largest tile in a corner
  • Build a chain of decreasing values
  • Plan ahead for each move
